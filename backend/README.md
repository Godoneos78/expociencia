# Backend – EXPOCIENCIA

Backend del sistema EXPOCIENCIA.

- Framework: Django
- Base de datos: PostgreSQL
- API: GraphQL (Graphene-Django)
- Objetivo: Exponer la lógica de negocio y datos para el frontend.

## Próximos pasos

- Crear entorno virtual (`venv`).
- Instalar dependencias base (Django, psycopg2-binary, graphene-django, django-cors-headers).
- Crear proyecto principal `config`.
- Crear apps: `usuarios`, `eventos`, `proyectos`, `evaluacion`, `premios`, `documentos`.
