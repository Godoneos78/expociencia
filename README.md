# EXPOCIENCIA – Sistema de Gestión de Eventos Académicos y Científicos

Este proyecto tiene como objetivo desarrollar una plataforma web para la gestión integral
de eventos académicos y de investigación (ferias científicas, expoemprende, etc.), incluyendo:

- Configuración de eventos, cronogramas y ofertas.
- Inscripción de participantes y registro de proyectos.
- Asignación de tribunales y evaluación mediante rúbricas.
- Gestión de premios y ganadores.
- Emisión de certificados con plantillas y membretes.
- Dashboard para análisis y futuras integraciones con IA.

## Estructura del proyecto

- `backend/`: API en Django + GraphQL + PostgreSQL.
- `frontend/`: Aplicación web en React + TypeScript (consume la API GraphQL).
- `docs/`: Documentación funcional, técnica y diagramas.

## Tecnologías principales

- Backend: Python, Django, GraphQL, PostgreSQL.
- Frontend: React, TypeScript, Apollo Client.
- Arquitectura: MVC, patrones Template y Estado.
